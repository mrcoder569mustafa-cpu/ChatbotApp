ChatBot App with Mern using GEMINI-API-KEY
# AI Chatbot MERN App

## Features
- User Login & Email Verification
- JWT Authentication
- AI Chat using Gemini API
- Chat History (MongoDB)
- Protected Routes
- Responsive UI with TailwindCSS

## Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB
- JWT
- Gemini API
- TailwindCSS

## frontend + backend setup
- npm install

## frontend + backend start commands
 - npm run dev
